# Kropp Fitness Landing Page

A modern, responsive landing page prototype for a fitness club website. This project uses clean HTML and CSS to present a stylish gym experience with promotional sections, training types, location details, and a BMI calculator.

## 🌐 Demo

[View Live Demo](https://hokkaidodev.github.io/kropp-fitness/)

## 🎨 Design Reference

[Figma Design Link](https://www.figma.com/design/m3lT3GF4mUgEBaU3qysyeu/10--Free-Web-UI-designs--Community-?node-id=0-1&p=f&t=Ta7cOCibu5ucDnlW-0)

## Features

- Responsive hero banner with event promotion
- Motivation cards with strong visual storytelling
- Training type showcase grid
- Video section with a call-to-action invite
- Join form for user subscriptions
- Location information and map imagery
- Gallery-style family section with scrollable images
- BMI calculator form and informational table
- Footer with contact details, working hours, and newsletter form

## Technology

- HTML
- CSS
- Static assets: images, icons, fonts

## Usage

Open `index.html` in a web browser to view the landing page.

## Notes

This project is a static front-end prototype designed for a fitness or gym brand. It does not include back-end functionality or JavaScript behavior for form submission or calculator logic.
